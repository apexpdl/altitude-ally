Altitude Ally is a mobile app built with MIT App Inventor for mountaineers. It helps you navigate trails, stay safe, connect with fellow climbers, and get AI-powered guidance.

Features
Altitude GPT: Ask anything about mountaineering and get AI-generated advice and information.

Compass: Navigate trails safely with an integrated compass.

AI Image Classifier: Detect potentially harmful animals and plants using AI-powered image recognition.

Friend Location Tracker: See your friends’ locations on the trail for safety and coordination.

Language Translator: Translate text in the app to your preferred language.

Mountaineer Chatrooms: Connect with other hikers, share tips, or report trail conditions.

AI Equipment Checklist: Learn about each piece of mountaineering gear with detailed AI-driven explanations.


Installation
Download the .aia project file from GitHub

Open MIT App Inventor and import the .aia file.

Connect your device or emulator and click Build → App (provide QR code for .apk).

Install the app and start your mountaineering adventure.


Usage
Open Home Screen to access Altitude GPT, compass, and chatrooms.

Snap a photo for the AI Image Classifier to identify animals or plants.

Track friends’ locations with Friend Tracker.

Learn about gear in AI Equipment Checklist.

Use the Language Translator for multilingual accessibility.


Technical Details
Platform: MIT App Inventor (Android)

AI Features: Image classification, Altitude GPT for mountaineering guidance

Sensors Used: Camera, compass, GPS
